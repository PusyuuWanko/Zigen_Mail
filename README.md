# Zigen_Mail

時限爆弾のような時限メールです。
nodejsがインストールされていない場合は
linux↓
ubuntuの例 → apt install nodejs

## 使い方
auto_mail.jsを開き自分のメールアドレスとパスワードの入力や、宛先に等の設定が完了したら以下のコマンドを入力してください。
1. npm install nodemailer
2. node auto_mail.js
